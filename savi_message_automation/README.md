# SAVI Message Automation

Phase 3 automation module for SAVI milestone communication.

## Included Automation

- Lead enquiry acknowledgement email.
- Dispatch update email on completed outgoing delivery.
- Engineer assignment activity on service assignment.
- Optional customer service assignment email.
- Optional automatic SRN feedback email on SRN completion.
- AMC renewal activity cron, with optional customer renewal email.

Customer-facing emails are disabled by default through settings. Internal service assignment and AMC renewal activities are enabled by default.
