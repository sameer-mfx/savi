HR Performance Review System

Complete review cycles with self-assessment, manager review, HR approval, goal tracking, and composite scoring

What This Module Does

A comprehensive performance review system supporting structured evaluation through multi-stage reviews. Includes weighted review criteria, self and manager assessments, goal tracking, composite scoring, and deadline-driven automation.

Key Features

  • Review Cycles — Quarterly, Semi-Annual, Annual, or Custom
  • Templates — Weighted criteria (must total 100%)
  • Self Assessment — Employees rate themselves with comments
  • Manager Review — Managers assess with their own scores
  • HR Approval — Final approval with computed ratings
  • Goal Tracking — Track goals with progress (0-100%)
  • Composite Rating — 70% Manager + 20% Self + 10% Goals
  • Deadline Reminders — Auto-email 3 days before deadlines

Workflow: Performance Review Cycle

Review States: Draft → Self Review → Manager Review → HR Review → Done
Rating Formula: 70% Manager Score + 20% Self Score + 10% Goal Achievement

Phase 1: Setup (HR Manager)

  1. Create a Template at HR → Performance → Templates:
    • Add criteria: Performance (30%), Competency (25%), Leadership (20%), Teamwork (15%), Innovation (10%)
    • Weights must total 100%
  2. Create a Review Cycle at HR → Performance → Review Cycles:
    • Set cycle type, date range, deadlines, template, and departments
  3. Click "Generate Reviews" → creates individual review records
  4. Click "Start" → sends all reviews to self-review stage with email notifications
Use Case Example — Annual Review Setup:
Cycle: "Annual Review 2026"
Period: Jan 1 – Dec 31, 2026
Self Review Deadline: Jan 15, 2027
Manager Review Deadline: Jan 31, 2027
Departments: All

Generate → 50 reviews created
Start → All employees receive: "Your performance review is ready for self-assessment"

Phase 2: Self Review (Employee)

  1. Employee opens their review from HR → Performance → My Reviews
  2. For each criteria: give a self-score (0-5) and add a self-comment
  3. Fill in: Self Summary, Strengths, Areas for Improvement
  4. Click "Submit Self Review" → moves to Manager Review

Phase 3: Manager Review

  1. Manager sees employee's self-scores and comments
  2. For each criteria: give a manager score (0-5) and manager comment
  3. Fill in: Manager Summary, Strengths, Development Areas
  4. Click "Submit Manager Review" → moves to HR Review

Phase 4: HR Approval

  1. HR reviews all scores and adds comments if needed
  2. Click "Approve"
  3. System computes final rating and sends completion notification
Use Case Example — Rating Calculation:
Employee: Suresh
  • Self-assessment weighted score: 3.8 / 5
  • Manager assessment weighted score: 4.2 / 5
  • Goal achievement rate: 85% (= 4.25 / 5)
  • Final Rating = (4.2 × 0.7) + (3.8 × 0.2) + (4.25 × 0.1) = 2.94 + 0.76 + 0.425 = 4.125 / 5 → "Exceeds Expectations"

Goal Tracking

  1. Create goals from the employee profile or review form
  2. Each goal has: Name, Type (Performance/Development/Project/Learning/Behavioral), Priority, Deadline, Progress (0-100%)
  3. Goal states: Draft → In Progress → Achieved / Partially Achieved / Not Achieved
  4. Average goal progress contributes 10% to the final rating

What NOT To Do

  • Don't create templates with weights ≠ 100% — System enforces this
  • Don't skip self-review — Manager review only opens after self-assessment is submitted
  • Don't rate criteria outside 0-5 — System validates score range
  • Don't set goal progress above 100%
  • Don't start a cycle without generating reviews first
  • Don't ignore deadline reminders — Auto-sent 3 days before each deadline
