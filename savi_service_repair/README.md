# SAVI Service Repair

Adds a structured repair case workflow for service operations:

- Link repair cases to service orders and helpdesk tickets.
- Track onsite/offsite repair.
- Capture issue, inspection, solution, new product discussion, and AMC discussion.
- Track estimate, advance payment, and customer approval.
- Attach issue photos, solution photos, and repair documents.
- Manage repair states from draft to done.
