|---|---|---|
| Core CRM/Sales/Purchase/Inventory/Accounting | Strong base coverage. Most standard transaction flows are available. | Medium: focus on configuration, adoption, and reports. |

| HR/Admin | Core HR, attendance, leave, payroll, performance review exist. Policy-heavy items need scope decisions before customization. | Medium |
| Statutory finance | Accounting/GST/payroll base exists. Government filing integrations are not evident. | Low-to-medium depending on compliance automation scope. |
